### About Subscriptions

Setup subscriptions paying in CashTokens for regular payments to a pre-defined address.

Subscribers can cancel at any time and withdraw the balance of the pre-paid fund using an NFT "key card".

More about the [contract here](https://bitcoincashresearch.org/t/unspent-phi-v3-timelocking-token-aware-contracts/1501#subscriptions-share-vesting-revocable-token-annuities-3)
