### About Locktime

Locktime allows locking coins (or CashTokens) until some future date.

**⚠️ NOTE: All assets remain LOCKED until the specified time, with no early withdraws possible.**

More about the [contract here](https://bitcoincashresearch.org/t/unspent-phi-v3-timelocking-token-aware-contracts/1501#absolute-timelock-bip65-hodl-style-vault-with-cashtoken-support-1)
